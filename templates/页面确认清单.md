# 页面确认清单

每设计一个页面，AI 先生成这张表。人只审批页面目标、主要按钮、敏感操作和高风险权限。

给 AI 的指令：

```text
请根据需求文档、数据字段和权限矩阵，为每个页面生成页面确认清单。
请先给出完整建议，并标出需要人确认的高风险点。
```

## 验收

- [ ] 页面第一屏能看出用途。
- [ ] 有清晰标题和主要按钮。
- [ ] 常用筛选不用隐藏太深。
- [ ] 表格字段能支持日常判断。
- [ ] 长文字不会重叠。
- [ ] 手机端可用。
- [ ] 无权限用户看不到敏感操作。
- [ ] 重要操作有确认和日志。

## 需要人确认

- [ ] 这个页面是否属于第一版。
- [ ] 是否显示敏感字段。
- [ ] 是否允许批量操作。
- [ ] 是否允许导出。
- [ ] 删除、覆盖、发送通知等操作是否需要二次确认。
