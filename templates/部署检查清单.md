# 部署检查清单

上线前逐项确认。

## 代码和构建

- [ ] 本地能启动。
- [ ] 构建命令成功。
- [ ] 没有明显 lint / type 错误。
- [ ] 关键页面浏览器检查通过。
- [ ] 没有真实数据、`.env`、数据库文件、密钥被提交。

## 环境变量

| 变量 | 用途 | 是否已配置 | 备注 |
|---|---|---|---|
| DATABASE_URL | 数据库连接 |  |  |
| AUTH_SECRET | 登录加密 |  |  |
| PUBLIC_SITE_URL | 站点地址 |  |  |
| EMAIL_API_KEY | 邮件服务，可选 |  |  |

## 数据库

- [ ] 正式数据库已创建。
- [ ] 迁移或建表已执行。
- [ ] 初始管理员账号已创建。
- [ ] 备份策略已确认。
- [ ] 恢复方法已记录。

## 上线后验证

- [ ] 登录可用。
- [ ] 首页看板有数据。
- [ ] 学员列表和详情可打开。
- [ ] 导入预览和错误提示可用。
- [ ] 报表可生成或导出。
- [ ] 权限拦截正确。
- [ ] 手机和电脑都能打开。
- [ ] HTTPS 生效。
- [ ] 管理员知道如何查看日志。
