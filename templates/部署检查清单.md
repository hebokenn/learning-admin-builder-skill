# 部署检查清单

上线前由 AI 先填写，人只确认授权、密钥、费用和正式发布。更严格的上线判断见 [部署前 Go/No-Go 检查表](部署前Go-No-Go检查表.md)。

## 代码和构建

| 检查项 | 检查方法 / 命令 | 负责人 | 证据 | 阻塞级别 | 结果 |
|---|---|---|---|---|---|
| 本地能启动 | 读取 `package.json` 后启动 | AI | 本地 URL / 截图 | 高 |  |
| 构建成功 | `npm run build` 或等价命令 | AI | 构建日志 | 高 |  |
| lint / typecheck | 项目已有命令 | AI | 命令输出 | 中 |  |
| 关键页面通过 | 浏览器检查 | AI | 页面列表 / 截图路径 | 高 |  |
| 安全扫描 | 查 `.env`、数据库、真实表格、密钥 | AI | 扫描结果 | 高 |  |

## 环境变量

| 变量 | 用途 | 来源 | 是否生产必需 | 是否已配置 | 备注 |
|---|---|---|---|---|---|
| DATABASE_URL | 数据库连接 | 部署平台 / 数据库平台 | 是 |  |  |
| AUTH_SECRET | 登录加密 | 部署平台密钥 | 是 |  |  |
| PUBLIC_SITE_URL | 站点地址 | 部署平台 / 域名 | 是 |  |  |
| EMAIL_API_KEY | 邮件服务，可选 | 邮件服务平台 | 否 |  |  |

## 数据库

| 检查项 | 检查方法 | 负责人 | 证据 | 阻塞级别 | 结果 |
|---|---|---|---|---|---|
| 正式数据库已创建 | 数据库平台查看 | 人授权 / AI 检查 | 数据库名称 | 高 |  |
| 迁移可执行 | 迁移命令 dry-run 或预演 | AI | 迁移日志 | 高 |  |
| 初始管理员方案 | seed 或管理后台创建 | 人授权 | 操作记录 | 高 |  |
| 备份策略 | 平台备份或脚本 | AI | 备份设置 | 高 |  |
| 恢复方法 | 恢复演练或步骤 | AI | 恢复说明 | 中 |  |
