# 02-工具和安装：小白准备清单

这一章告诉你制作培训进度网站需要哪些工具。不要害怕工具多：你不需要一开始全部精通，只要知道每个工具负责什么。

## 推荐工具总表

| 类别 | 推荐工具 | 用途 | 官方链接 |
|---|---|---|---|
| AI 助手 | ChatGPT | 问需求、写提示词、解释错误 | [ChatGPT](https://chatgpt.com/) |
| AI 编程 | OpenAI Codex | 读代码、改代码、运行检查、提交 Git | [Codex 文档](https://developers.openai.com/codex/cloud) |
| AI 编辑器 | Cursor | 在代码编辑器里边看边改 | [Cursor](https://cursor.com/) |
| 代码编辑器 | Visual Studio Code | 打开项目、查看文件、编辑代码 | [VS Code](https://code.visualstudio.com/) |
| 版本管理 | Git | 保存每次改动，方便回退 | [Git 文档](https://git-scm.com/doc) |
| 代码托管 | GitHub | 存放代码、公开教程、部署连接 | [GitHub Docs](https://docs.github.com/en) |
| 运行环境 | Node.js | 运行 Next.js 网站和脚本 | [Node.js 下载](https://nodejs.org/en/download) |
| 网站框架 | Next.js | 做前后端一体的网站 | [Next.js Docs](https://nextjs.org/docs) |
| 数据库 | PostgreSQL | 正式保存学员、课程、进度 | [PostgreSQL Docs](https://www.postgresql.org/docs/) |
| 托管数据库 | Supabase / Neon | 小白更容易用的在线 Postgres | [Supabase](https://supabase.com/docs)、[Neon](https://neon.com/docs) |
| 部署平台 | Vercel / Render | 把网站发布到互联网 | [Vercel](https://vercel.com/docs)、[Render](https://render.com/docs) |
| 服务器进程 | PM2 | VPS 上保持网站一直运行 | [PM2](https://pm2.keymetrics.io/docs/usage/quick-start/) |
| 反向代理 | Nginx | VPS 上绑定域名、转发访问 | [Nginx](https://nginx.org/en/docs/) |
| HTTPS | Certbot | 给域名配置 HTTPS 证书 | [Certbot](https://certbot.eff.org/) |

## 小白应该先装哪些

如果你刚开始，只需要：

1. ChatGPT 或 Codex
2. GitHub 账号
3. VS Code 或 Cursor
4. Node.js
5. Git

先不要急着买服务器。第一版可以先在本地电脑跑起来，再部署。

## 安装后怎么检查

打开终端，输入：

```bash
node -v
npm -v
git --version
```

如果能看到版本号，说明基础环境可用。

如果报错，把完整错误复制给 AI：

```text
我在安装培训进度网站开发工具时遇到错误。
我输入的命令是：
【粘贴命令】

终端返回：
【粘贴错误】

请用小白能懂的话告诉我原因和下一步。
```

## 部署平台怎么选

| 选择 | 适合情况 | 不适合情况 |
|---|---|---|
| 本地预览 | 只给自己看、确认页面和流程 | 正式给团队使用 |
| Vercel | Next.js 网站、快速上线、少维护 | 复杂后台任务、需要完整服务器控制 |
| Render | 需要长期运行服务、后台任务、数据库服务 | 极度依赖自定义服务器环境 |
| VPS 云服务器 | 要完整控制、Nginx、PM2、定时脚本、私有部署 | 没有人懂服务器维护 |

培训管理类网站如果有真实学员数据，推荐至少做到：

- 登录保护。
- 数据库定期备份。
- HTTPS。
- 管理员和普通用户权限区分。

## 不建议一开始做的事

- 不要一开始就买很贵的服务器。
- 不要把真实学员数据传到公开仓库。
- 不要把数据库密码写在代码里。
- 不要还没确认页面和数据就让 AI 写一堆代码。
- 不要所有管理员共用一个账号。
