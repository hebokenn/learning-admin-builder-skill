## 导入页

```text
┌────────────────────────────────────────────────────────────┐
│ 数据导入                                      [下载模板]    │
├────────────────────────────────────────────────────────────┤
│ 选择数据类型：学员名单 / 进度更新 / 成绩 / 证书              │
│ 上传文件：[选择 Excel/CSV]                                   │
├────────────────────────────────────────────────────────────┤
│ 字段映射                                                     │
│ Excel 字段：姓名 -> 系统字段：learner.name                   │
│ Excel 字段：工号 -> 系统字段：learner.employee_id            │
├────────────────────────────────────────────────────────────┤
│ 预览结果：共 120 行，可导入 117 行，错误 3 行                │
│ 错误行：第 18 行 邮箱格式错误；第 44 行 工号重复             │
│ [重新上传] [确认导入]                                        │
└────────────────────────────────────────────────────────────┘
```
